# magic-device-tool
A batch for installing Ubuntu and Android on your device

Currently supported devices:

- BQ Aquaris E4.5 (krillin)
- BQ Aquaris E5 HD (vegetahd)
- Meizu MX 4 (arale)
- LG Nexus 4 (mako)
- Asus Nexus 7 (flo)

Usage
-----
```
$ chmod +x launcher.sh
$ ./launcher.sh
```
